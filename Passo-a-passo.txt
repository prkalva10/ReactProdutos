Projeto de Front React - Produtos:

1. Criar o repositório no GIT;

2. Criar o projeto:
    npm create vite@latest ReactProduto

3. Comandos para instalar algumas bibliotecas e teste inicial:
    cd ReactProduto
    npm install axios react-router-dom bootstrap
    npm run dev

4. Limpar o conteúdo dos arquivos App que vem com um template;

5. Adicionar o bootstrap no main.jsx:

    import 'bootstrap/dist/css/bootstrap.min.css';

6. Criar Componentes:
    - Menu:
    - Grid;
    - Form;

7. Ajustar as rotas;

8. Layout do grid;

9. DB por JSON:
    - Criar o JSON de dados;
    - Instalar o json server:
    npm install -g json-server

10. Servidor json server:
    json-server --watch db.json --port 5000

11. Commit desses passos iniciais:

    git config --global user.name "<meu-usuario-git>"
    git config --global user.email "<meu-email-git>"

    git commit -m "feat: inicia projeto ReactProduto"
    git push -u origin main

12. Criar botões de edição e excluir no grid;

13. Criar formulário de edição;

14. Funcionamento do editar;

15. Funcionamento do salvar, tanto para inserção quanto para atualização;

16. Funcionamento do Excluir;

17. Início das melhorias - colocar uma janela de confirmação de exclusão mais bonita;
